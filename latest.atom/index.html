Latest Blog Posts http://localhost/latest.atom/ 2017-01-22T12:00:00Z http://localhost/ Werkzeug mendeleev web app http://localhost/news/2016-12-13-mendeleev-web-app/ 2017-01-22T12:00:00Z 2017-01-22T12:00:00Z <p>Since I first discovered <a href="http://bokeh.pydata.org/en/latest/">bokeh</a> I instantly adopted it to my work flow. Currently I use it mainly for visualizing intermediate results in <a href="http://jupyter.org/">Jupyter notebooks</a> when doing data explorations or code experiments. The interactive tools it offers are extremely helpful especially with larger sets of data and ones containing outliers where visual inspection can speed up the whole analysis.</p> <p>While reading various bits of <a href="http://bokeh.pydata.org/en/latest/">bokeh documentation</a> I discovered that <a href="http://bokeh.pydata.org/en/latest/">bokeh</a> offers a lot more that just <a href="http://jupyter.org/">Jupyter</a> support but actually provides excellent tools for creating interactive visualizations for for the web including plots, widgets etc.</p> <p>At that time I was already working on my <a href="http://mendeleev.readthedocs.io/en/stable/">mendeleev</a> python package which collects considerable amounts of data on elements from reliable source and wraps it in a convenient <a href="https://www.python.org/">python</a> API. Therefore when I saw one of the examples from the <a href="http://bokeh.pydata.org/en/latest/docs/gallery.html">bokeh gallery</a> that used the library to <a href="http://bokeh.pydata.org/en/latest/docs/gallery/periodic.html">display the periodic table</a> I started thinking about a web app.</p> <p>The result is the <a href="https://mendeleev.herokuapp.com">mendeleev app</a> which allows interactive explorations of the periodic table and relationships between various properties of the elements.</p> <h2><a href="https://mendeleev.herokuapp.com/periodic">Periodic Table</a></h2> <p>The <a href="https://mendeleev.herokuapp.com">main page</a> allows creating custom made periodic tables with a variety of properties displayed next to the standard symbol, name and atomic number. The <strong>Property</strong> can be chosen from the drop down list and will be displayed in the bottom part of each rectangle that represents an element. Moreover some basic properties will be displayed by the <a href="http://bokeh.pydata.org/en/latest/docs/user_guide/tools.html#hovertool">HoverTool</a> when you hover over the element.</p> <p>The second list named <strong>Color by</strong> contains a list of common categories that elements can belong to, such as: block, period, series and so on. Selecting one of those categories will apply the appropriate color mapping to the elements. In the example below the elements have distinct colors for different series. </p> <div class="row" style="padding-top: 30px;padding-bottom: 30px"> <a href="https://mendeleev.herokuapp.com/periodic/?prop=mass&colorby=name_series&palette=Viridis256"> <img class="img-responsive center-block" width=700px src="../../static/img/mendeleev_periodic_series.png"> </a> </div> <p>The last entry from the <strong>Color by</strong> list is <strong>Property</strong> which applies a color map (based on the palette selected from the third list <strong>Palettes</strong>) to the selected property resulting in a heat map - type plot. The example below shows such a heat map applied to the values of covalent radius (Pyykko) where the <a href="http://matplotlib.org/examples/color/colormaps_reference.html">Inferno</a> palette was selected. The largest values are the brightest - yellowish color for Cs and Fr and the smallest values are the darkest - see H and He.</p> <div class="row" style="padding-top: 30px;padding-bottom: 30px"> <a href="https://mendeleev.herokuapp.com/?prop=covalent_radius_pyykko&colorby=property&palette=Inferno256"> <img class="img-responsive center-block" width=700px src="../../static/img/mendeleev_periodic_cov_rad_property.png"> </a> </div> <p>If you would like to save a png file with a particular view of the periodic table you can use the <a href="http://bokeh.pydata.org/en/latest/docs/user_guide/tools.html#savetool">bokeh save tool</a> in the upper right corner of the plot. </p> <h2><a href="https://mendeleev.herokuapp.com/correlations">Correlations</a></h2> <p>Another useful feature of the app can be found under the <a href="https://mendeleev.herokuapp.com/correlations">correlations</a> tab, where you can produce <a href="https://en.wikipedia.org/wiki/Scatter_plot">scatter plots</a> and explore the relations between various properties of the elements.</p> <p>The main motivation for this part of the app came from the observation that there are many variations of the periodic tables on the web, however most of them are focused on exposing information about individual elements rather that uncovering the interdependencies, regularities and laws relating different elements.</p> <p>To update the plot select what should be on the <strong>X Axis</strong> and the <strong>Y Axis</strong> from the lists and additionally choose what category should be used to color map the points. Below is an example in which the covalent radius (by Pyykko) is plotted against the atomic number with series as the categorical variable to guide the colors. </p> <div class="row" style="padding-top: 30px;padding-bottom: 30px"> <a href="https://mendeleev.herokuapp.com/correlations/?x=atomic_number&y=covalent_radius_pyykko&categ=name_series"> <img class="img-responsive center-block" width=700px src="../../static/img/mendeleev_corr_an_vs_cov_rad_series.png"> </a> </div> <p>If you go to the <a href="https://mendeleev.herokuapp.com/correlations/?x=atomic_number&amp;y=covalent_radius_pyykko&amp;categ=name_series">actual application</a> you can interact with the plot even further by using the <a href="http://bokeh.pydata.org/en/latest/">bokeh</a> toolbar located in the upper right corner. You can pan the plot, zoom on a particular area, resize the plot and save a resulting png image. Moreover if you put your mouse cursor over any point the <a href="http://bokeh.pydata.org/en/latest/docs/user_guide/tools.html#hovertool">HoverTool</a> will give you additional information about the element the point corresponds to.</p> <p>Another example below illustrates how ionization energy changes with covalent radius (by Pyykko) categorized by the block:</p> <div class="row" style="padding-top: 30px;padding-bottom: 30px"> <a href="https://mendeleev.herokuapp.com/correlations/?x=covalent_radius_pyykko&y=ionization_energy&categ=block"> <img class="img-responsive center-block" width=700px src="../../static/img/mendeleev_corr_cov_rad_vs_ie_block.png"> </a> </div> <h2>Get Involved</h2> <p>The app uses <a href="http://flask.pocoo.org/">flask</a> as the framework binding all the parts together and internally <a href="http://pandas.pydata.org/">pandas</a> as the source of the data structures. If you are interested where the raw data comes from check out the <a href="http://mendeleev.readthedocs.io/en/stable/data.html">data documentation of mendeleev</a> where you can find all the details.</p> <p>If you have any suggestions about the app or you found a bug you can let me know by creating a <a href="https://github.com/lmmentel/mendeleevapp/issues">github issue</a> or simply leave a comment below. </p> <p>If you would like to help by contributing some code or documentation, that's great.
